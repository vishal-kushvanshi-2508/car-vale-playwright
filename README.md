# CarVale Scraper Using Playwright

## 📖 Overview

This project is a Python-based web scraping solution built using Playwright to extract car-related information from dynamic websites. The scraper automates browser interactions, handles JavaScript-rendered content, and collects structured vehicle data efficiently.

The project demonstrates browser automation, data extraction, error handling, and scalable scraping techniques used in real-world applications.

---

## 🚀 Features

* Browser automation using Playwright
* Dynamic website scraping
* Car information extraction
* Structured data collection
* JSON data processing
* Error handling and logging
* Clean and maintainable project structure

---

## 🛠️ Technologies Used

* Python
* Playwright
* JSON
* Requests
* Logging

---

## 📊 Extracted Data

* Car Name
* Brand
* Model
* Variant
* Price
* Specifications
* Product URL
